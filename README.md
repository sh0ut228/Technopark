# MAX Bot Platform

Платформа для создания бота в мессенджере MAX с древовидным меню и веб-админкой.

## Возможности

- 🤖 **MAX Bot** - бот с древовидным меню поддержки бизнеса
- 📊 **SQLite Database** - хранение ссылок, пользователей и статистики
- 🖥️ **Web Admin Panel** - управление ботом через веб-интерфейс
- 📈 **Статистика** - отслеживание переходов и активности пользователей
- 🔒 **Аутентификация** - защищенный доступ к админке

1. Установка зависимостей:
bash
pip install -r requirements.txt
2. Настройка .env:
env
MAX_BOT_TOKEN=ваш_токен_от_MasterBot
3. Инициализация базы данных:
bash

4. Запуск бота (в отдельном терминале):
bash
python -m bot.max_bot
5. Запуск админ-панели (в отдельном терминале):
bash
uvicorn web.main:app --reload
6. Запуск тестов:
bash
